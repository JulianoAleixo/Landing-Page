Meus melhores projetos

Aqui irei falar dos projetos apresentados na feira de tecnologia ProjETE promovida pela ETE FMC

Sun Mind - 2022

Um dispositivo responsável pelo processo automático de termografia de usinas fotovoltaicas. Consiste em um carrinho de controle remoto controlado por Bluetooth a partir de uma Raspberry Pi 3 B+, um ESP8266 para o acionamento do motor, uma webcam simulando uma camera térmica e uma rede neural para que possa ser analisada as fotos posteriormente. Foram utilizadas bibliotecas do Python para se comunicar com o celular pelo Bluetooth (Bluez), para tirar as fotos (OpenCV) e para a rede neural (TensorFlow e Keras).

Remember - 2021

Um dispositivo voltado para o uso de pessoas com o estágio inicial de Alzheimer ou para pessoas que consumam muita medicação. É um disco que serve o remédio correto na hora correta e informa ao dono caso o remédio tenha sido tirado do dispenser. O disco foi moldado e impresso em 3d. Foram usados um sensor foto-voltáico para analizar se o remédio foi retirado ou não. A comunicação do dispositivo acontece por Wi-Fi pelo ESP8266 e os dados são trocados entre o aplicativo, que possui um banco de dados.

Sk8 Vibration - 2020

Uma simulação de um skate eletrônico. A simulação foi feita utilizando a plataforma Tinkercad. Foi utilizado um sensor LDR para identificar se havia alguem sobre o skate, para que assim possa ser acionado o motor. Além disso, o skate possui 2 modos de uso, iniciante e experiente e 2 informações a serem mostradas no display LCD, velocidade e tempo de uso.
